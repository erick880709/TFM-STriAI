3. Instalación Paso a Paso

1 Abrir una terminal

Windows: PowerShell o CMD. macOS/Linux: Terminal.

2 Navegar a la carpeta del proyecto

PS> cd "C:\Users\ELITEBOOK\OneDrive\Documentos\Repositorio\Trabajo\TFM-FINAL"

3 Crear el entorno virtual

PS> python -m venv .venv

Esto crea una carpeta .venv/ con un Python aislado para el proyecto.

4 Activar el entorno virtual

Windows PowerShell:

PS> .\.venv\Scripts\Activate.ps1
⚠️ Si aparece error de permisos: Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

macOS / Linux:

$ source .venv/bin/activate

Verás (.venv) al inicio del prompt indicando que el entorno está activo.

5 Instalar dependencias

(.venv) PS> pip install -r sistema-triaje-ia/requirements.txt

⏱️ Tiempo estimado: 5-15 minutos. PyTorch es el paquete más grande (~122 MB).

6 Configurar variables de entorno

(.venv) PS> Copy-Item sistema-triaje-ia\.env.example sistema-triaje-ia\.env

Esto copia la plantilla de configuración. No es necesario modificar nada para desarrollo.

7 Verificar instalación

(.venv) PS> python -c "import streamlit; import pandas; import sklearn; import xgboost; import torch; import transformers; print('✓ Todas las dependencias OK')"

Deberías ver: ✓ Todas las dependencias OK

8 Iniciar la aplicación

(.venv) PS> streamlit run app.py --server.port 8501

9 Abrir el navegador

Ve a http://localhost:8501

Usuario: admin | Contraseña: admin123

10. Solución de Problemas

10.1 Errores Frecuentes

❌ Python not found

Descargar Python desde python.org. ✅ Marcar "Add Python to PATH" durante la instalación.

❌ pip is not recognized

PS> python -m pip install -r sistema-triaje-ia/requirements.txt

❌ No module named 'streamlit'

Asegúrate de que el entorno virtual está activo (.venv) y reinstala las dependencias.

❌ WinError 206 — Ruta demasiado larga

Habilitar rutas largas en Windows (como Administrador):

PS (Admin)> New-ItemProperty -Path "HKLM:\SYSTEM\CurrentControlSet\Control\FileSystem" -Name "LongPathsEnabled" -Value 1 -PropertyType DWORD -Force

❌ Port 8501 already in use

PS> streamlit run app.py --server.port 8502

❌ Error de política de ejecución (Windows)

PS> Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

❌ OutOfMemoryError

El modelo NLP consume ~1.5 GB de RAM. Cerrar otras aplicaciones. La app funciona sin el modelo IA.

❌ bcrypt no se instala

Instalar Visual C++ Build Tools.
